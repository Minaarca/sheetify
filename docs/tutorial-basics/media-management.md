Export and optimize product media in bulk.

## 📤 Export Media

- Media ID

- File Type
- Linked Product
- File Size (Bytes & MB)

`Media`

| Column | Required |
|--------|----------|
| Product ID | Yes |
| Media URL | Yes |

If new → Add new media
